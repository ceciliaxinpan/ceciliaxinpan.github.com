Teacher Dashboard

An A teacher dashboard that generate learner data and performance

The dashboard is created for teachers who are using Alien Rescue, a 3D-based immersive game for sixth grade science. It intends to gathers, analyzes, and reports learner data on multiple dimensions, including computer-mediated user activity, such as time spent on tools and frequencies, student learning artifacts, such as learners’ post and feedback.
It is characterized as a visual display of the necessary information collected from students’ learning activities. Information is consolidated and arranged on a single screen, so teachers can come up with a direct understanding of student performances and teachers were able to keep track of student performance, manage activities, and give feedback in a more efficient way
